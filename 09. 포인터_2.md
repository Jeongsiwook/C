### void형 포인터
   - 자료형을 안만들고 쓰는 포인터
   ```
   #include <stdio.h>
   
   int main()
   {
       int num = 10;
       int *numPtr1 = &num1;
       void *ptr; // void 포인터
       ptr = numPtr1; // void 포인터에 int 포인터 저장
       numPtr1 = ptr; // int 포인터에 void 포인터 저장
       
       return 0;
   }
   ```
   - 역참조를 사용할 수 없음
      - 자료형이 없어 메모리의 크기를 얼만큼 자를 지를 모름
   - void형 변수도 만들 수 없음
